Blinx Project
May 2022 - December 2022,
May 2023
The blinx project complements the CodeBoot project .
The aim of the CodeBoot Project is to create a web application that lets you do programming, especially Python, from a browser, without any client-side installation.
The blinx project is the integration of microcontrollers into the web application. Thanks to the web serial api available on Google Chrome (as well as on Edge, Opera, chromium base browser).
The aim is to be able to control a microcontroller (the blinx) from the codeboot web application.
The project saw the light of day with the "school reform" at CEGEP, where programming become a requirement.
My supervisor wanted to offer CodeBoot to CEGEP as a learning tool. And Blinx, came as an extension of the possibilities, especially for the project required.
I was supervised by Marc Feeley, professor at UdeM.
