Hackathon SPI
10 - 14 January 2022
We presented the Garambullo Project the Hackathon given by CIMAT at the event SPI. To resolve recipe grouping/classification problems.
The aim of the Garambullo Project is to create a bot that will return a recipe based on the message sent by the user.
However, it is possible that several recipes match the user's request.
Since only a certain number can be sent back, we need to classify them to find out which one matches the user's needs the most.
This is where we needed help.
