Création d'un LSP pour le language P4.
A DevOps framework for developing, operating, and evolving SDN infrastructure and applications.
Septembre 2021
Le but du projet été de créer un LSP pour le language P4, avec un gestionnaire de plugin intégré pour pouvoir ajouter des scripts à exécuter à certains moments. Notamment, des tests (P4test ou Petr4) à éxécuter à chaque sauvegarde de fichier.
LSP permet de faire du : syntax highlighting, code refactoring (e.g, renaming), syntactical diagnostic, code completion and navigation (e.g., “go to definition”).
