Creation of an LSP for the P4 language.
A DevOps framework for developing, operating, and evolving SDN infrastructure and applications.
September 2021
The aim of the project was to create an LSP for the P4 language, with an integrated plugin manager to add scripts to be executed at certain times. For example, tests (P4test or Petr4) to be run each time a file is saved.
LSP allows you to : syntax highlighting, code refactoring (e.g, renaming), syntactical diagnostic, code completion and navigation (e.g., “go to definition”).
